# 🧮 CALCIFY - Modern Web Calculator

CALCIFY is a sleek, glassmorphic calculator application designed for a premium user experience. It combines a minimalist aesthetic with robust functionality, including keyboard support and advanced CSS animations.

## ✨ Features

- **Futuristic UI**: Features a dark-mode interface with radial gradients and neon accents.
- **Dynamic Display**: The input text automatically shrinks (auto-scaling) as numbers get longer to prevent layout breaking.
- **Micro-interactions**: Includes ripple effects on button clicks and 3D "rise" animations on load.
- **Keyboard Support**: Fully functional using your physical keyboard (numbers, operators, Enter, and Backspace).
- **Responsive Design**: Built using CSS Grid and Flexbox for a consistent experience on all screen sizes.
- **Smart Logic**: Handles floating-point precision (e.g., 0.1 + 0.2) and prevents division by zero.

## 🚀 Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Grid, Flexbox, and Keyframe animations.
- **Vanilla JavaScript**: State management and event delegation.
- **Google Fonts**: 'DM Mono' for that technical feel and 'Syne' for bold headings.
